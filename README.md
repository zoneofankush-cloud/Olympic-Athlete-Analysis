# Olympic Athlete Performance Analysis

## 📌 Project Overview
This project analyzes Olympic athlete participation and performance trends using historical Olympic data from **1896–2016**. The objective is to extract actionable insights using SQL-based analytics, cloud data warehousing, and interactive business intelligence dashboards.

## 📂 Repository Structure
* **dataset/**: Raw CSV files.
* **sql_queries/**: SQL scripts used for analysis in BigQuery.
* **dashboards/**: Power BI file and screenshots.
* **presentation/**: PowerPoint slides.
* **architecture/**: Data pipeline diagram.

## 📊 Key Insights
* **Total Athletes:** 271,000+
* **Gender Gap:** Female participation has increased significantly after 1980.
* **Top Sports:** Athletics, Gymnastics, and Swimming have the highest participation.
* **Medal Dominance:** USA, Russia, and Germany lead the medal counts.

## 🛠 Tools Used
* **SQL (BigQuery)** for data analysis.
* **Power BI** for visualization.
* **PowerPoint** for presentation.
